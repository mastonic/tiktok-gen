## Comment nous avons automatisé la production
Pour déployer BentoPDF via Docker, commencez par récupérer l'image officielle. Utilisez la commande `docker pull bentopdf/bentopdf` pour instantanément obtenir la dernière version. Configurez votre environnement avec des paramètres appropriés pour un fonctionnement fluide.

L’automatisation peut être intégrée grâce à des scripts Docker Compose, garantissant une mise en route rapide. Cette méthode permet de maintenir une version à jour tout en simplifiant le processus de déploiement. Ainsi, le travail devient plus efficient sans sacrifier la qualité.
