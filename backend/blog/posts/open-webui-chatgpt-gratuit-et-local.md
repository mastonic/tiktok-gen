## Comment nous avons automatisé la production
Pour installer Open WebUI sur votre laptop, commencez par vous assurer que vous avez Ollama déjà configuré. Ouvrez votre terminal et exécutez la commande suivante : 

```bash
git clone https://github.com/username/open-webui.git
cd open-webui
npm install
```

Une fois les dépendances installées, il vous suffit de lancer l'interface avec `npm start`. Voilà, votre ChatGPT local est prêt à interagir, sans que vous ayez besoin de passer par des serveurs distants ou de traiter des dizaines d’abonnements mensuels. La beauté de l’open source réside dans sa simplicité et sa transparence.

Pour les utilisateurs plus avancés, vous pouvez personnaliser le contexte de la conversation en modifiant les fichiers de configuration dans le dossier `config`. Bien sûr, si quelque chose tourne mal... n'hésitez pas à ouvrir un ticket sur GitHub, car le support est aussi local que l'installation.
